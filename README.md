# Android Movie App


## Tech/ framework used

* Android Jetpack MVVM
* DI with Hilt
* Android Navigation Graph
* Live Data
* Picasso Image Loading
* Retrofit Network
* Carousel View

## API Reference
[The Movie Databse API](https://developers.themoviedb.org/3/getting-started/introduction)
